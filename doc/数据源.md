# 回放  
通过通用上位机软件记录的数据带有时间戳，可以通过回放的方式加载历史数据，并实现回放控制。

# 文件  
可打开存有原始数据的文件加载其数据，与回放不同的是，原始数据文件没有时间戳，是立刻加载的。所以曲线中的数据没有横坐标，需要在软件中通过指令配置横坐标的索引数据。
